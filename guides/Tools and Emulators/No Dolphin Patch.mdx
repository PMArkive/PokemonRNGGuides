## Tools

- A GBA emulator, such as [mGBA](https://mgba.io/downloads.html#desktop-os-1)
- patch link here

## Install the patch

1. Download the patch.
2. Place patch in same directory as multiboot rom
3. rename RS save to name of multiboot rom
4. load multiboot rom and redeem the Pokemon
5. Rename save back to what it was previously
